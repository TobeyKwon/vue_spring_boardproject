## 1. 프로젝트 개요
- 프로젝트 명칭 : 스프링부트, 뷰 게시판
- 개발 기간 : 2022.06.21 ~
- 개발 환경 : Vue.js [버전], SpringBoot [버전], gradle [버전] ~
- 데이터베이스 : MySql

## 2. 기능 명세
### 회원 기능
  - 회원가입
  - 회원수정
  - 회원정보 확인
  - 회원탈퇴
  - 로그인
### 게시판 기능
  - 글 작성
  - 글 목록
  - 글 상세페이지
  - 글 수정
  - 글 삭제
### 댓글 기능
  - 댓글 작성
  - 댓글 수정
  - 댓글 삭제
  - 대댓글(추가기능)
